# NOIR925 PWA Icons

This directory should contain the following icon files for the PWA:

## Required Icons
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Additional Icons (Optional)
- shop-icon.png (96x96) - For shop shortcut
- new-icon.png (96x96) - For new arrivals shortcut
- wishlist-icon.png (96x96) - For wishlist shortcut

## Icon Design Guidelines
- Background: Dark (#1E1E1E)
- Primary element: "N" or "925" in silver/white
- Style: Minimalist, luxury feel
- Safe zone: 10% padding from edges for maskable icons
